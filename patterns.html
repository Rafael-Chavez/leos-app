Pattern Guide
Counting by 1s
1, 2, 3, 4, 5...
Add 1 each time
Counting by 2s
2, 4, 6, 8, 10...
Add 2 each time (even numbers)
Counting by 5s
5, 10, 15, 20, 25...
Add 5 each time
Counting by 10s
10, 20, 30, 40, 50...
Add 10 each time
Counting Backwards
10, 9, 8, 7, 6...
Subtract 1 each time

How to Find Patterns

1. Look at the first two numbers
2. Find the difference between them
3. Check if the same difference continues
4. Use that pattern to find the missing number!
